## type field_t

```
type field_t = struct{
    string name
    int hash
    int offset
}
```

Represents a struct field with name, hash and offset information

## type type_t

```
type type_t = struct{
    int size
    int hash
    int kind
    u8 align
    [int] hashes
    [field_t] fields
}
```

Represents type information including size, hash, kind and field details

```
}
```
